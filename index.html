This is Flatpak-Testdrive - a Flatpak repository

This is a Flatpak repository for testing purposes. It contains some apps including a version history to test certain aspects of these apps.

Add to software manager

Flatter is ideal for a low-traffic Flatpak repository, like distributing automated nightly builds. A .flatpakrepo file was generated for this repository, making it easy for users to add this repo to their software manager.
